Skill cloud API:

The Skill cloud API allows you to search jobs with specified skills with JSON over HTTP.


To get a directory of skills with their IDs, use this URL:

http://ec2-18-222-28-147.us-east-2.compute.amazonaws.com/api/v1/ref/skills/


To get a list of jobs for a particular skill, use this URL:

http://ec2-18-222-28-147.us-east-2.compute.amazonaws.com/api/v1/vacancies/{skill}


To get a list of the 20 most required skills for a particular position, use this URL:

http://ec2-18-222-28-147.us-east-2.compute.amazonaws.com/api/v1/topskills/{position}
